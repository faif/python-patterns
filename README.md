python-patterns
===============

A collection of design patterns implemented (by other people) in python.

When an implementation is added or modified, be sure to update this file and
rerun `append_output.sh` to keep the output comments at the bottom up to date.

Current Patterns:

* 3-tier
* abstract_factory
* adapter
* borg
* bridge
* builder
* catalog
* chain
* command
* composite
* decorator
* facade
* factory_method
* flyweight
* graph_search
* iterator
* mediator
* memento
* mvc
* null
* observer
* pool
* prototype
* proxy
* publish_subscribe
* state
* strategy
* template
* visitor
